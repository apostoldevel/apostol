Апостол Веб Сервис

Апостол Веб Сервис (Apostol Web Service) - это комплекс технических решений собранных воедино для создания бэкэнд (англ. backend) части веб-приложений.

Апостол Веб Сервис состоит из сервера приложений и бэкэнд модуля:

  • Сервер приложений (HTTP-сервер) используется для удалённого вызова процедур с использованием архитектурного стиля REST (от англ. Representational State Transfer — "передача состояния представления").

  • Бэкэнд модуль используется для разработки бизнес-логики и RESTful веб-API созданного на базе AWS проекта. Адаптирован под разработку документо-ориентированных автоматизированных систем (CRM-систем).

После установки[1] Апостол Веб Сервис Вы получите "из коробки":

  • Сервер приложений (RESTful) разработанный на программной платформе Апостол;

  • Сервер авторизации разработанный на основе OAuth 2.0 и OpenID Connect;

  • Бэкэнд модуль разработанный под СУБД PostgreSQL;

  • API для взаимодействия с бэкэнд (с возможностью подключения по WebSocket);

  • Веб-сервер с минимальными функциями (аутентификация и авторизация пользователей);

  • Документацию к API для фронтэнд (англ. frontend) разработчиков (пример).

Останется лишь приступить к разработке фронтэнд части проекта на любом удобном для веб-разработчика языке программирования (PHP, JavaScript, Python и д.р) и бизнес логики на языке программирования PL/pgSQL (на котором написан бэкэнд модуль).

1. Поставляется и устанавливается на компьютер (сервер) в виде системной службы под операционную систему Linux.

Апостол

Апостол - это программная платформа для создания системного программного обеспечения.

Платформа написана на языке программирования C++, имеет модульную конструкцию и включает в себя встроенную поддержку СУБД PostgreSQL.

В руках опытного разработчика это мощный инструмент для создания веб приложений любого уровня сложности.

LAMP ➫ LAPP

AWS меняет парадигму разработки Web-приложений.

LAMP:

  • Linux — ОС Linux;
  • Apache — веб-сервер;
  • MariaDB/MySQL — СУБД;
  • PHP/Perl/Python — язык программирования;

LAPP:

  • Linux — ОС Linux;
  • Apostol — веб-сервер;
  • PostgeSQL — СУБД;
  • PL/pgSQL  — язык программирования;

Где Apostol, PostgeSQL и PL/pgSQL работают вместе как единый программный комплекс.

Сервер авторизации

Встроенный сервер авторизации разработан на основе спецификации OAuth 2.0 RFC 6749 и расширении OpenID Connect.

Сервер авторизации позволяет упростить доступ пользователей к приложениям, разрешив для этого использование существующей учетной записи социальной сети или, например, учетной записи пользователя в Google или в ЕСИА (Госуслуги).
